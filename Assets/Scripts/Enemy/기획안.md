# 게임 기획서: (가제) 아케이드 클라이머 (Arcade Climber)

## 1. 게임 개요 (Game Overview)
* **장르:** 3D 클라이밍 (암벽등반) / 플랫포머 / 서바이벌
* **핵심 컨셉:** 3D 공간의 플레이어가 고전 2D 아케이드 게임 세계가 구현된 거대한 벽을 등반하며, 각 게임의 고유한 방해 요소를 피해 꼭대기 층에 도달하는 게임.
* **시각적 특징:** 입체적인 3D 플레이어 캐릭터와 평면적이고 레트로한 2D 픽셀 게임 벽면의 시각적 대비.

---

## 2. 핵심 시스템 (Core Mechanics)

### 2.1. 3D-2D 상호작용 클라이밍
* 플레이어는 3D 물리 법칙을 따르며 벽을 타고 오르는 클라이밍을 수행한다.
* 벽면을 구성하는 2D 게임의 지형지물과 상관없이 플레이어는 자유롭게 클라이밍을 수행한다.

### 2.2. 차원간 충돌 및 회피 (3D vs 2D)
* 맵에 등장하는 적과 장애물은 철저히 2D 게임의 규칙(그리드 이동, 지정된 패턴, 2D 물리)에 따라 움직인다.
* 3D 플레이어는 벽에 매달린 상태에서 이 2D 방해 요소들과 충돌하면 피해를 입거나 추락(사망)하므로, 타이밍과 동선을 계산하여 회피해야 한다.

---

## 3. 레벨 디자인: 층별 기믹 (Level Design)

### [1층] 팩맨 (Pac-Man) 구역: 패턴 인식과 추격 회피
* **맵 환경:** 어두운 배경에 네온 라인으로 구성된 팩맨 미로 구조의 벽면.
* **방해 요소 (유령):** 2D 유령들이 미로의 선을 따라 직진 및 교차로 이동 패턴을 보이며 순찰한다. 
* **플레이 포인트:** * 유령은 3D 공간으로 튀어나오지 않고 2D 벽면 내에서만 움직인다.
  * 플레이어는 유령이 지나가는 타이밍을 피해 이동해야 한다. 접촉 시 플레이어는 즉사한다.

### [2층] 동키콩 (Donkey Kong) 구역: 타이밍과 궤적 예측
* **맵 환경:** 붉은 철골 구조물과 사다리가 지그재그로 배치된 공사장 테마의 벽면.
* **방해 요소 (술통):** 상단에서 지속적으로 2D 술통이 생성되어 철골 경사를 따라 굴러떨어지며, 무작위로 사다리를 타고 수직 낙하하기도 한다.
* **플레이 포인트:**
  * 굴러오는 술통을 피해 등반해야 한다.

### [3층] 테트리스 (Tetris) 구역: 지형 변화와 순발력
* **맵 환경:** 하단에 일부 블록이 쌓여있고, 상단에서 끊임없이 새로운 블록이 내려오는 거대한 격자형 벽면.
* **방해 요소 및 기믹 (테트로미노):**
  * 위에서 아래로 떨어지는 2D 테트리스 블록에 깔리면 플레이어는 추락하거나 사망한다.
* **플레이 포인트:**
  * 떨어지는 장애물을 피해야 한다.
